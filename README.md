# 心理咨询服务平台 · 大学实验项目
## 项目简介

心理咨询网站系统是一个基于Spring + SpringMVC + MyBatis框架开发的Web应用系统，旨在为大学生提供心理健康服务。系统集成了心理测评、在线咨询、文章阅读、社区交流等功能，为学校心理健康教育提供数字化支持。

一个基于Spring MVC与MyBatis的心理咨询预约系统，支持用户线上咨询、专家预约、情绪自评与管理工具

🏆 项目亮点
​​高校心理咨询解决方案​​：针对大学生心理健康场景定制化开发
​​多角色用户体系​​：学生用户/咨询师/管理员三级权限设计
​​隐私保护机制​​：咨询记录加密存储与访问控制
​​响应式前端​​：兼容PC与移动设备访问
​​心理评估工具​​：集成SDS抑郁量表、SAS焦虑量表

## 功能特性

### 🧠 心理测评模块
- **多种测评量表**：支持焦虑自评量表(SAS)、人际关系测评、大学生心理健康测试等多种专业量表
- **智能评分系统**：根据测评结果自动计算得分并生成专业建议
- **测评记录管理**：完整记录用户测评历史，支持查看历史结果
- **个性化建议**：根据得分区间提供针对性的心理建议

### 👨‍⚕️ 在线咨询模块
- **医生信息管理**：展示专业心理医生信息，包括资质、专长等
- **预约咨询**：学生可在线预约心理咨询服务
- **留言交流**：支持学生与医生之间的留言交流
- **预约确认**：医生可确认或调整预约时间

### 📚 心理知识库
- **心理美文**：提供心理健康相关的文章阅读
- **心理手册**：专业的心理健康指导手册
- **文章分类**：按主题分类管理心理知识内容
- **收藏功能**：用户可收藏感兴趣的文章

### 💬 情感驿站
- **社区交流**：学生可发布帖子分享心理感受
- **评论互动**：支持帖子评论和回复功能
- **多媒体支持**：支持图片、视频等多媒体内容
- **用户互动**：促进用户之间的交流和支持

### 📢 公告管理
- **重要通知**：发布心理健康相关的重要通知
- **活动宣传**：宣传心理健康教育活动
- **信息发布**：及时发布心理健康相关信息

### 👤 用户管理
- **多角色支持**：支持学生、医生、管理员等多种角色
- **个人信息管理**：用户可管理个人基本信息和头像
- **权限控制**：基于角色的访问权限控制

## 技术架构

### 后端技术栈
- **框架**：Spring 4.2.5 + SpringMVC + MyBatis 3.4.6
- **数据库**：MySQL 5.7+
- **连接池**：Apache Commons DBCP
- **日志**：Log4j + SLF4J
- **JSON处理**：Jackson 2.7.1
- **构建工具**：Maven

### 前端技术栈
- **页面技术**：JSP + JSTL
- **UI框架**：Layui
- **富文本编辑器**：UEditor
- **轮播组件**：Swiper.js
- **图片上传**：原生JavaScript

### 开发环境
- **JDK版本**：Java 8
- **服务器**：Tomcat 8+
- **数据库**：MySQL 5.7+
- **IDE推荐**：IntelliJ IDEA / Eclipse

## 安装部署

### 环境准备
1. **Java环境**：安装JDK 8或以上版本
2. **数据库**：安装MySQL 5.7或以上版本
3. **服务器**：安装Tomcat 8或以上版本
4. **IDE**：推荐使用IntelliJ IDEA或Eclipse

### 数据库配置
1. 创建数据库：
```sql
CREATE DATABASE bishe DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

2. 导入数据：
```bash
mysql -u root -p bishe < database/shujuku.sql
```

3. 修改数据库连接配置：
编辑 `src/resources/jdbc.properties` 文件：
```properties
url=jdbc:mysql://localhost:3306/bishe?useUnicode=true&characterEncoding=UTF-8&useOldAliasMetadataBehavior=true
username=root
password=your_password
```

## 常见问题

### 1. 数据库连接失败
- 检查数据库服务是否启动
- 确认数据库连接配置是否正确
- 检查数据库用户名和密码

### 2. 页面显示异常
- 检查Tomcat日志查看错误信息
- 确认JSP文件路径是否正确
- 检查静态资源文件是否存在

### 3. 文件上传失败
- 检查upload目录权限
- 确认文件大小是否超限
- 检查文件类型是否允许
